# 🚀 AI Job Copilot

AI-powered job application assistant that helps users optimize resumes, generate cover letters, and track job applications — all in one platform.

---

## ✨ Features

- 🔐 Authentication (Signup, Login, Logout, Forgot Password)
- 📄 Resume Analysis (AI-powered insights)
- 🎯 Job Match Scoring
- 📝 Cover Letter Generator
- 📊 Application Tracker (Full CRUD with Supabase)
- 🌙 Dark Mode Support

---

## 🛠 Tech Stack

- **Frontend:** Next.js (App Router), Tailwind CSS
- **Backend:** Supabase (Auth + Database)
- **AI Integration:** OpenAI (optional / mock-ready)
- **Deployment:** Vercel (recommended)
